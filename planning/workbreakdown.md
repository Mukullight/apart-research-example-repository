# Work Breakdown Structure (WBS): AI Safety Fellowship (Apart Research Model)

**Project:** AI Safety Fellowship – Pilot Cohort  
**Date:** 2026-06-11  
**Version:** 1.0  

---

## WBS Hierarchy (3 levels)

**Level 1:** `1. AI Safety Fellowship Pilot`

**Level 2 (Phases) & Level 3 (Work Packages)**

| WBS Code | Work Package Name | Description | Owner |
|----------|-------------------|-------------|-------|
| **1.1 Program Setup & Adaptation** | | | |
| 1.1.1 | Adapt Apart Research Playbook | Customise curriculum, seminar templates, and mentor guidelines for our cohort. | Program Lead |
| 1.1.2 | Mentor Recruitment & Contracts | Identify, invite, and contract 8 mentors. | Mentorship Coordinator |
| 1.1.3 | Operational Tool Setup | Configure Zoom, Slack, Notion/Airtable, GitHub for cohort use. | Ops Coordinator |
| 1.1.4 | Legal & Compliance Review | Finalize fellow/mentor agreements and stipend payment process with fiscal sponsor. | Fiscal Sponsor / Legal |
| 1.1.5 | Stipend Tier Design | Define tier structure (e.g., $500–$2,000) based on region and need. | Program Lead |
| **1.2 Outreach & Selection** | | | |
| 1.2.1 | Application Portal & Rubric | Build application form (Airtable/Google Forms) and scoring rubric. | Ops Coordinator |
| 1.2.2 | Marketing & Promotion | Social media, community posts, partner outreach to attract applicants. | Marketing Volunteer / Ops |
| 1.2.3 | Application Review & Interviews | Screen, score, and conduct short interviews; finalise 15 fellows. | Selection Committee |
| **1.3 Fellowship Execution & Support** | | | |
| 1.3.1 | Onboarding & Orientation | Welcome pack, platform walkthrough, mentor matching. | Program Lead |
| 1.3.2 | Weekly Seminars & Co‑working | Host 12 weekly sessions with invited speakers or facilitated discussions. | Program Lead / Mentors |
| 1.3.3 | Mentorship Sessions | Schedule and support 1‑on‑1 mentor meetings (1h/week per fellow). | Mentorship Coordinator |
| 1.3.4 | Community Management | Moderate Slack, foster peer support, share resources. | Community Manager (volunteer) |
| 1.3.5 | Stipend Disbursement | Process monthly tiered stipend payments to fellows. | Finance/Ops |
| 1.3.6 | Mentor Honoraria & Support | Pay honoraria, gather mentor feedback, handle issues. | Program Lead |
| **1.4 Evaluation & Close‑out** | | | |
| 1.4.1 | Mid‑program Check‑in | Survey fellows and mentors; adjust as needed. | Program Lead |
| 1.4.2 | Final Symposium | Organise online showcase of fellow research projects. | Program Lead |
| 1.4.3 | Final Report & Funding Proposal | Evaluate pilot, compile metrics, draft report and next‑cohort proposal. | Program Lead |

## WBS Dictionary Example
| WBS: 1.3.5 | Stipend Disbursement |
|-------------|-----------------------|
| Description | Pay 15 fellows monthly based on pre‑defined tiers via Wise or similar. |
| Deliverables | Payment confirmations, expense records. |
| Acceptance Criteria | 100% on‑time payments; all transactions compliant with fiscal sponsor rules. |
| Assumptions | Fellows provide correct bank details by Week 1. |
